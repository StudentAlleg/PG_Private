## How to Use These Documents

**For a warm introduction:**
Send the Executive Summary or One-Pager with the intro email.

**For a first meeting:**
Present the Pitch Deck (transfer content to slides first). Demo the product live using the Demo Script.

**For due diligence:**
Share the Investor Memo and Financial Model. These provide the depth that serious investors need.

**For internal planning:**
Use the Fundraising Strategy, Legal Prerequisites, and Investor Targets to manage the process.
