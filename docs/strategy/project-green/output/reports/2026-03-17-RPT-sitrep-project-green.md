## Five Questions and Chairman's Answers

### 1. Owen Status and Capacity

**Question:** Owen is at 5 hrs/week, job searching, and skeptical. What's his actual capacity over the next 90 days?

**Chairman's answer:** Owen should take a full-time job if he finds one. His 25% stake is earned through contribution, not guaranteed. If employed elsewhere, Owen should be "overemployed" — keeping Project Green as an entrepreneurial endeavor alongside his day job. Hours may fluctuate but the expectation is continued effort proportional to the stake.

**Operational implication:** Plan around variable Owen hours. Do not create dependencies on Owen availability for critical-path work.

### 2. Epic 34 — Go or Hold?

**Question:** The gold layer can't reliably answer the three alpha questions. Fix first or test to see what breaks?

**Chairman's answer:** Go. Run the test with CSV file upload (bypass QBO API). Test with real Delray Dermatology data.

**Operational implication:** Prepare Epic 34 sprint plan with CSV upload path. Use the test to identify gold layer gaps empirically rather than theoretically.

### 3. QBO API Blocker

**Question:** Is the QBO API still blocking?

**Chairman's answer:** Development application submitted to Intuit, awaiting approval. No longer blocking — CSV export loader built 3/16 serves as bypass. QBO API approval becomes an upgrade when it arrives, not a gate.

**Operational implication:** QBO API removed from critical path. Track Intuit approval as a parallel workstream.

### 4. Parr3 Path Read

**Question:** Which path are we heading toward — collaborative (Path 1) or separation (Path 2)?

**Chairman's answer:** Expects a win/win (Path 1). Stays employed, gets IP acknowledgement, gets the executive role he's been asking for, work becomes dual-purpose (Project Purple at Parr3 + Project Green independently), and launches his company simultaneously. If it goes the other way, he's at peace and prepared.

**Operational implication:** Plan for Path 1 as primary. Maintain Path 2 readiness without letting it slow Path 1 execution.

### 5. Burn Rate and Runway

**Question:** How many months can you sustain self-funding?

**Chairman's answer:** If employed — indefinitely. The $750-$1,200/month business burn is sustainable against $160K combined income. If separated, he solves it entrepreneurially — first revenue, consulting, advisory work. He feels more equipped for that than at any prior point in his life.

**Operational implication:** No artificial runway deadline while employed. If separated, first revenue becomes the #1 priority.

## Alexander's Operational Takeaways

1. **Plan for Path 1, prepare for Path 2.** Don't let downside planning slow upside execution.
2. **Epic 34 sprint plan** — CSV upload with real Delray Derm data. Identify gold layer gaps empirically.
3. **Dual-track roadmap** — timeline differences between Path 1 (employed + PG) and Path 2 (full-time founder).
4. **Owen engagement framework** — what "earning the 25%" looks like at various hour commitments.
