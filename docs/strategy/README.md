# Project Green -- Strategy

The complete strategic operating plan for Project Green. Each document is self-contained but designed to be read as a set.

## Reading Order

**Start here:** [Executive Strategy Brief](executive-strategy-brief.md) -- the thesis, the bet, the objectives.

Then by audience:

- **Investors / advisors:** Market Landscape, Market Gap Analysis, Financial Framework, Risk Register
- **Product / engineering:** Product Strategy, Technical Architecture, Market Gap Analysis
- **Sales / growth:** Go-to-Market Playbook, Traction Acceleration, Demo Playbook, AI Readiness Assessment
- **Market research:** Market Gap Analysis, Market Research Gaps, GTM Evaluation
